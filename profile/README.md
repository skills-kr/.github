# GitHub 여정을 시작하세요

<img alt="" src=https://user-images.githubusercontent.com/1221423/156894097-ff2d6566-7b6a-4488-950e-f4ebe990965a.svg width=200 align=right>

_초보자와 전문가를 위해 설계된 대화형 실습으로 GitHub 사용법을 배워보세요._

- **재미있는 학습**: 시뮬레이션이나 지루한 튜토리얼 없이, GitHub이 만들고 Issues와 GitHub Actions를 활용한 실전 레슨입니다.
- **실제 프로젝트**: 실제 프로젝트의 복사본에서 새로운 기술을 배웁니다.
- **친절한 안내**: Octocat이 여정 전체에 걸쳐 안내와 피드백을 제공합니다.
- **실제 워크플로우**: Issues, Actions, Codespaces 등 실제 GitHub 기능으로 진행됩니다.

[🏆 GitHub Skills Workshop Dashboard](https://github-skills.studydev.com) 에서 실습 진행 상황을 실시간으로 확인하세요!

팀에서 실제로 사용하는 도구로 나만의 실습 콘텐츠를 만들어 보세요. 🌟  

> :copilot: Copilot을 활용해 실습을 자동으로 생성하는 방법을 단 10분 만에 배울 수 있습니다.  
[Skills Exercise Creator](https://github.com/skills/exercise-creator) 가이드를 확인해 보세요!
